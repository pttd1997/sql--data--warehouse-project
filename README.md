# sql-data-warehouse-project

Welcome to the SQL Data Warehouse Project!

Building a modern data warehouse using SQL Server, including ETL processes, data modeling and analytics.

This repository documents my hands-on learning journey building a modern data warehouse from scratch using SQL Server,  covering ETL processes, dimensional data modeling, and analytics best practices.

The project follows real-world data engineering standards, including naming conventions and architectural patterns used in production environments at large organizations. It's designed as a portfolio piece that bridges the gap between learning and industry-ready work.

🎓 Built by following the excellent tutorial by Baraa Khatib Salkini - a data expert with 10+ years of experience leading Big Data, Data Lakehouse, and BI projects at top companies including Mercedes-Benz. Huge thanks for making this content freely available!
